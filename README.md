## DataPulse Lab

Projeto de engenharia e análise de dados simulando o ambiente de uma clínica de saúde, com foco em qualidade de dados, automação e governança.

Este projeto está sendo desenvolvido como prática de:

- SQL e modelagem relacional
- Pipeline de dados (ETL) com Python
- APIs com FastAPI
- Validação e qualidade de dados
- Visualização e dashboards
- Boas práticas de organização e documentação

### Objetivo

Construir uma plataforma de dados completa que simule um ambiente real de clínica de saúde, permitindo:

- ingestão de dados de pacientes, consultas e exames
- limpeza e validação automática de dados
- detecção de anomalias
- monitoramento da qualidade dos dados
- geração de relatórios e dashboards
- rastreabilidade e auditoria de alterações

### Tecnologias Utilizadas

- Python
- FastAPI
- MySQL
- SQL
- Pandas
- Git e GitHub
